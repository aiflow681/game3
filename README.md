【游戏名称】户外滑雪

【游戏介绍】
这是一款H5原生开发的3D动作类小游戏。游戏采用Three.js引擎开发，提供流畅的3D滑雪体验。玩家控制一个滑雪者角色，在雪山环境中进行滑雪运动，需要躲避沿途的松树障碍物，尽可能滑得更远获得更高分数。游戏具有真实的物理引擎和流畅的动画效果。

【玩法说明】
操作：
- 使用屏幕上的虚拟方向按钮控制滑雪者移动
- 点击"←"按钮向左移动
- 点击"→"按钮向右移动
- 点击"🔄"按钮重新开始游戏
- 也可以使用键盘的方向键进行控制
- 使用W键向前滑行，A/D键或左右方向键控制左右转向

规则：
- 滑雪者需要躲避沿途的松树障碍物
- 一旦碰到松树，游戏结束
- 滑雪速度会随着时间自动增加
- 滑雪者可以左右转向躲避障碍物
- 持续滑雪可以获得更高的分数

目标：
- 在雪山环境中尽可能长时间地滑雪
- 躲避所有松树障碍物
- 获得最高的滑雪距离分数
- 挑战自己的反应速度和操作技巧

Tips：
- 提前观察前方路况，规划好滑行路线
- 适当左右转向，不要一直保持直线
- 速度越快时，转向要更加谨慎
- 观察松树的分布模式，找出安全通道
- 保持冷静，不要在障碍物前慌乱转向

技术介绍：
- 使用HTML5 Canvas和JavaScript开发
- 物理引擎模拟真实的滑雪动力学
- 响应式设计，完美适配移动端和桌面端
- 虚拟控制器专为移动设备优化，提供流畅的触摸操作
- 高性能渲染系统，支持复杂的光照和阴影效果
- 动画混合系统，实现平滑的角色动作转换
